
# Описания проекта к КТ2 для программных проектов

## 1. Основные планы и этапы проекта

### 1.1 Краткое описание проекта:  
**Название проекта**: Система распознавания PDF и изображений для преобразования данных в структурированный формат — серверная часть  
**Цель проекта**: Обеспечить автоматическую обработку PDF и изображений с последующим распознаванием текста, анализом структуры и экспортом в структурированные форматы через REST API  
**Краткое описание задач**: Использование OCR-движка, структурного анализа, классификации документов, хранения результатов и предоставления API для интеграции  

### 1.2 Планы и этапы выполнения проекта:

| Этап проекта                     | Описание работ                                                 | Ожидаемые результаты                             | Сроки выполнения     |
|----------------------------------|----------------------------------------------------------------|--------------------------------------------------|----------------------|
| Эскизный проект                  | Проектирование архитектуры API и базы данных                   | Спецификация API, ER-диаграмма БД                | 17.12.25 – 10.01.26  |
| Технический проект               | Разработка модулей с использованием OCR и структурного анализа | Работающие модули Tesseract + OpenCV             | 11.01.26 – 28.02.26  |
| Рабочий проект                   | Интеграция компонентов и настройка очередей задач              | Единая серверная система с поддержкой batch-обработки | 12.03.26 – 31.03.26  |
| Рабочий проект                   | Подготовка документации                                        | Архитектурная схема, OpenAPI, руководство по развёртыванию | 11.04.26 – 30.04.26  |

## 2. Используемый технологический стек и его обоснование

### 2.1 Перечень используемых технологий:

| Технология/Инструмент | Описание                                                   | Причины выбора                                               |
|-----------------------|------------------------------------------------------------|--------------------------------------------------------------|
| Python 3.9+           | Язык программирования                                      | Поддержка научных библиотек, простота интеграции OCR        |
| Flask 3.1+            | Веб-фреймворк                                              | Надёжность, готовая система REST, инструменты для API       |
| PostgreSQL 13+        | Система управления базами данных                          | Надёжность, поддержка JSON, совместимость с Django          |
| Tesseract OCR 5.0+    | Движок оптического распознавания                          | Открытый, поддержка русского языка, высокая точность       |
| OpenCV 4.5+           | Библиотека компьютерного зрения                            | Предобработка изображений, выделение таблиц                 |
| Redis 6.0+ + Celery   | Очереди задач и кэширование                                | Асинхронная обработка тяжёлых задач (OCR)                   |
| Docker 20.10+         | Контейнеризация                                            | Упрощённое развёртывание, изоляция зависимостей            |

### 2.2 Обоснование выбранного технологического стека:  
Выбранный стек обеспечивает полную локальную обработку документов без зависимости от облака. Flask предоставляет готовую основу для REST API и управления задачами. Python в связке с Tesseract, OpenCV — стандарт в задачах OCR и структурного анализа. PostgreSQL гарантирует надёжное хранение структурированных и полуструктурированных данных. Redis + Celery — общепринятое решение для масштабируемой фоновой обработки.
