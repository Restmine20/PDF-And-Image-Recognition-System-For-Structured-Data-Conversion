
# Описания проекта к КТ2 для программных проектов

## 1. Основные планы и этапы проекта

### 1.1 Краткое описание проекта:  
**Название проекта**: Система распознавания PDF и изображений для преобразования данных в структурированный формат — клиентская часть  
**Цель проекта**: Обеспечить интуитивный веб-интерфейс для загрузки документов, просмотра и корректировки результатов распознавания, а также экспорта в структурированные форматы  
**Краткое описание задач**: Разработка минималистичного, с поддержкой drag & drop, предпросмотра, редактирования и управления API-ключами

## 2. Используемый технологический стек и его обоснование

### 2.1 Перечень используемых технологий:

| Технология/Инструмент           | Описание                                                   | Причины выбора                                               |
|----------------------------------|------------------------------------------------------------|--------------------------------------------------------------|
| TypeScript 5.0+                  | Язык программирования                                      | Статическая типизация, надёжность кода                      |
| React 18+                        | Фреймворк для UI                                           | Компонентный подход, широкая экосистема                     |
| React Context                    | Управление состоянием                                      | Достаточно для масштаба проекта, без избыточных зависимостей|
| Vitest + React Testing Library   | Тестирование                                               | Современные, быстрые, интегрированы с Vite                  |

### 2.2 Обоснование выбранного технологического стека:  
TypeScript + React — стандарт для современных веб-приложений, обеспечивающий надёжность и поддерживаемость.

## 4. Особые пометки

- API-ключи могут храниться в localStorage по выбору пользователя  
- Все данные передаются по HTTPS  
- Интеграция с Django через REST API с заголовком X-API-Key
